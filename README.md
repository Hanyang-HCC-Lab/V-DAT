# V-DAT

## V-DAT frontend

### Start frontend server

```
cd V-DAT\ frontend
npm install
npm run serve
```

## V-DAT backend

### Requirements

We use [Azure anomaly detection](https://azure.microsoft.com/en-us/products/cognitive-services/anomaly-detector/) 

```
pip install -r requirements.txt
```

### Start backend server

```
cd V-DAT\ backend
python server.py
```
